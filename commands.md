- [x] `HELLO`

## String (KV)
- [x] `GET`
- [x] `SET` (options: `NX`, `XX`, `EX`, `GET`, `IFEQ`)
- [x] `DEL`
- [x] `EXISTS`
- [x] `INCR`
- [x] `DECR`
- [x] `DECRBY`

## Expiry
- [x] `TTL`
- [x] `PTTL`

## List
- [x] `LPUSH`
- [x] `RPUSH`
- [x] `LPOP`
- [x] `RPOP`
- [x] `LLEN`
- [x] `LINDEX`
- [x] `BLPOP`
